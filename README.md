# Casa Conceito Decor

Landing page de uma loja de decoração desenvolvida com HTML, CSS e JavaScript puro, simulando uma vitrine de produtos de uma loja de decoração sofisticada.

O projeto tem como objetivo criar uma experiência moderna, elegante e responsiva para apresentação de produtos, contato com clientes e pesquisa por CEP utilizando a API viaCep.

![Casa Conceito Decor](https://github.com/user-attachments/assets/26601848-e5df-4c61-ab6b-ec6b57a3fc57)

## Tecnologias

- HTML5
- CSS3
- JavaScript ES6+
- Web Components
- ViaCEP API
- Google Fonts
